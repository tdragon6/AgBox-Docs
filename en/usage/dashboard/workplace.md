# Workplace
This page displays the overall operation status of the AgBox OPC platform.
<Image src="/AgBox-Docs/images/usage/dashboard/workplace/workplace.png" />

## Data Statistics
The data statistics section shows the count information of existing digital employees, skill library, tasks, and model configurations.

## Model Consumption
The model consumption section displays the model consumption of all or each digital employee (including synchronous and asynchronous coordinators). You can switch views via tabs.
<Image src="/AgBox-Docs/images/usage/dashboard/workplace/workplace_robots_cost.png" />

## Quick Links
The right side provides quick access links to `Repository Address`, `Employee Marketplace`, and `Usage Documentation`.
