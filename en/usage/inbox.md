# Inbox
This page receives notifications for task completion and digital employee upgrades.
<Image src="/AgBox-Docs/images/usage/inbox/inbox.png" />

## Field Description
Refer to [Field Description - Inbox](/en/usage/key#inbox)

## Interface Layout
- The left side displays all digital employees/coordinators that have notification messages and their corresponding unread message counts.
- The right side shows the detailed notification information list of the selected digital employee/coordinator.

## Search & Sort
- Supports searching by `Notification Type`, `Task/Upgrade Status`, `Task Message`, `Whether Task is Read`, `Creation Time`, and `Update Time` fields. `Notification Type` and `Task/Upgrade Status` support multiple selection.
- Supports sorting by `Notification Type`, `Task/Upgrade Status`, `Whether Task is Read`, `Creation Time`, and `Update Time` fields.

## View Messages
- `Task Completion` type messages display task results.
<Image src="/AgBox-Docs/images/usage/inbox/inbox_result.png" />
- `Employee Upgrade` type messages display employee upgrade details.
<Image src="/AgBox-Docs/images/usage/inbox/inbox_upgrade.gif" />
- After viewing a message, the corresponding message will be marked as read.

## Delete Messages
Supports deleting/batch deleting notification messages.

## Mark All as Read
After marking all as read, all messages of the current digital employee/coordinator will be marked as read.
